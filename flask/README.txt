Some nice commands to keep in mind:
    * venv\Scripts\activate
    * flask run [options: --host=0.0.0.0]
    * flask shell
    * set FLASK_DEBUG=1 [or 0]

Default website can be found by following the provided links:
    * http://localhost:5000/index
    * http://127.0.0.1:5000/
